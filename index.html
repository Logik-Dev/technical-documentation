Introduction

CSS est un language de programmation qui n'est jamais cité seul et que l'on associe généralement à HTML, et pour cause il n'a pas d'utilité sans ce dernier.En effet CSS sert à styliser les pages web écrites en HTML, avec CSS on peut par exemple:

  • Changer la police d'un texte.
  • Changer la couleur de fond d'un élément.
  • Modifier la hauteur ou la largeur d'un élément.
  • Créer des animations, etc...

Mais on peut encore faire tellement d'autre choses avec CSS qu'en faire une liste serait bien trop long et rébarbatif, mais c'est un language très facile d'accès et le seul requis est de connaître HTML.

Syntax

La syntaxe de CSS est très simple.Voici par exemple comment séléctionner une balise titre <h1> et changer sa couleur en rouge: 

h1 { color: #f00; }

Comme on modifie généralement plus d'une propriété on aura plutôt tendance à utiliser la notation sur plusieurs lignes: 

h1{
    color: #f00;
    font-size: 20px;
}

Içi on a simplement modifier la couleur de tout les titres <h1> en rouge ainsi que la taille de leur police à 20 pixels.

Sélecteurs

Un sélecteur peut être de différent type:

  • Le nom d'une balise.
  • Le nom d'une classe avec un point le précédent.
  • Le nom d'un id avec un dièse le précédent.

Par exemple pour changer la couleur de fond de mon élément qui à la classe bg-red il me suffit d'ajouter un point devant le nom de la classe: 

.bg-red{
    background-color: #f00;
}

Combinateurs

On peut combiner de différentes façon les sélecteurs pour pouvoir sélectionner plus finement nos éléments.Le symbole + permet de selectionner le prochain élément qui a le même parent: 

h1 + p{
    font-size: 15px;
}

Içi on sélectionne tout les premiers paragraphes situés juste après une balise <h1>.

Le symbole > permet lui de selectionner les enfants d'un élément, par exemple on peut sélectionner tout les liens contenus dans un paragraphe et modifier leur affichage par défaut: 

p > a{
    text-decoration: none;
    color: inherit;
}
